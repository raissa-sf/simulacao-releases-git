## Sistema Web de Login — Simulação de Releases com Git
---

## 🎯 Objetivo do Projeto

O objetivo deste projeto é aplicar conceitos de:

- Controle de versão com Git
- GitHub Flow e gerenciamento de branches
- Pull Requests e revisão de código
- Releases e versionamento
- Trabalho colaborativo em equipe

A aplicação consiste em um sistema web simples de login desenvolvido utilizando HTML e JavaScript.

## 🌳 Estratégia de Branches

| Branch | Finalidade |
|---|---|
| `main` | Versão estável |
| `develop` | Integração das funcionalidades |
| `feature/*` | Novas funcionalidades |
| `fix/*` | Correções |
| `release/*` | Preparação de releases |

### Exemplos

```bash
feature/login
feature/admin-page
fix/validacao-usuario
release/v1.0
```

1. Clone o repositório:

2. Abra o projeto no VS Code.

3. Execute o arquivo `index.html` no navegador.
